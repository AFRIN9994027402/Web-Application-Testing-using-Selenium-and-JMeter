# Web Application Testing using Selenium and JMeter

## 📌 Project Description
This project demonstrates how to perform functional and performance testing on a web application using Selenium and Apache JMeter. Selenium is used for automating browser-based functional test cases, while JMeter is used to measure performance metrics such as response time, throughput, and error rate.

## 🎯 Objectives
- Automate functional test cases using Selenium
- Integrate Selenium tests with JMeter
- Analyze performance under load conditions
- Identify failures and bottlenecks in the application

## 🛠️ Tools & Technologies Used
- Java
- Selenium WebDriver
- Apache JMeter
- TestNG / JUnit
- ChromeDriver

## 🚀 Project Workflow
1. Create test scripts using Selenium
2. Execute functional test cases
3. Integrate Selenium scripts with JMeter
4. Run load testing using JMeter
5. Analyze results (response time, success rate, etc.)

## 📊 Results
- Successfully executed automated test cases
- Measured application performance under different loads
- Identified response delays and failure scenarios
